# CTS系统接口对接验证报告

## 1. 系统整体架构概述

CTS（Container Transfer System）是一个完整的容器镜像自适应分发系统，采用模块化设计，包含以下核心组件：

### 1.1 核心模块组件
- **环境感知模块** (`EnvironmentController`)：实时探测网络与算力资源状态
- **模型包装器** (`CFTNetWrapper`)：CFT-Net性能预测模型的封装接口
- **决策引擎** (`CAGSDecisionEngine`)：基于预测结果的风险感知决策系统
- **执行器** (`DockerExecutor`)：容器镜像拉取与管理系统

### 1.2 实验框架组件
- **实验编排器** (`run_all_experiment.py`)：总控实验执行流程
- **测试床** (`testbed/`)：Docker容器化测试环境
- **实验脚本** (`experiments/`)：五个核心实验模块
- **结果分析器**：实验数据分析与可视化
